How I did a Regex Crossword

This is a Regular Expression crossword that we did in class. The way this works is that every box has 4 different regular expressions that it must match. Let's go over one example before the full answer

I'm going to be focusing on the top left box as an example.

The first expression is .*[GAF]*, which searches for any number of any characters, then searches for any number of either a capital G, A, or F.

The second expression is [ONE]*[SKA], which searches for any number of either O, N, or E and then searches for one of either S, K, or A.

The third expression is (NF|FA|A|FN)+, which searches for one or more of those groups.

The last expression is .*(O|S)*, which searches for zero or more of anything then searches for zero or more of either O or S

This is the solved crossword that comes out to be 'NOSFERATU', a famous vampire from old movies.
