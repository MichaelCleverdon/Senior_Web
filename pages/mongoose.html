How I got Mongoose installed and how I used it in a MERN Stack application

This is the first step I took to install Mongoose and some other tools to my app.

After installing the packages through NPM, I initialized the package I will be using for my app with NPM init.

Then I created my app.js file for node to read. In this file, I require the different packages I installed earlier like Mongoose, Express, and Body-Parser. Also, I require a model called Book, which is created from the schema in another file called Book.Model.js

This is the Book.Model.js, which defines the schema I am going to use for this database.

As you can see, this file requires Mongoose, which is necessary to create schema. This specific schema requires that Title be a string, Author be a string, and Category be a string. After defining what the schema will be, Mongoose exports the schema as a model called 'Book'

Going back to the app.js file, you can see it connects to the database named 'example' on my local machine. Then, using Express, I can get the http request and the response, which I can then use to put "Happy to be here" on my webpage.
